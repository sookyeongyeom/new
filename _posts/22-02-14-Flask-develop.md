> FLASK_APP

- Linux/Mac

```bash
$ export FLASK_APP=myapp
```
- Windows CMD

```bash
$ set FLASK_APP=myapp
```

<br>

> FLASK_ENV

- Linux/Mac

```bash
$ export FLASK_ENV=development
```

- Windows CMD

```bash
$ set FLASK_ENV=development
```
