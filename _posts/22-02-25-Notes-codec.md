> 코덱이란?

코덱 = Coder + Decoder

비디오/오디오 저장 시 압축을 거쳐 저장됨

압축 과정 = 인코딩 → 코더

압축 풀기 = 디코딩 → 디코더

- 인코딩 : 파일을 압축, 변환하는 과정

- 디코딩 : 인코딩된 파일을 재생하는 과정

- 코더 : 파일을 인코딩해줌

- 디코더 : 파일을 디코딩해줌

인코딩된 파일에 맞는 디코더가 없다면 그 파일은 재생하지 못함


<br>

> 코덱 없을 때 해결 방법

1. 컴퓨터에 있는 디코더에 맞게 파일을 변환 → 이게 편함! (트랜스코딩=인코딩)

2. 파일에 맞는 디코더를 찾아 다운로드

- 압축된 파일을 필요한 형태로 바꾸는 것을 Transcoding(=Encoding)이라고 함
